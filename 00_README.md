## How to Use These Notes

1. **Read linearly** — each module builds on the previous.
2. **Don't skip Module 2** (SOLID) — it's the lens through which all design patterns make sense.
3. **For interview prep** — Modules 8–12 are the most directly useful; revisit patterns while solving them.
4. **Diagrams** — every module has Mermaid class/flow diagrams; render in VS Code with the Markdown Preview Mermaid Support extension.
